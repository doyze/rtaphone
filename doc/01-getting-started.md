# เริ่มต้นใช้งาน RTAphone

## ภาพรวม

RTAphone เป็นแอปพลิเคชันโทรศัพท์ผ่านอินเทอร์เน็ต (VoIP) ที่พัฒนาขึ้นสำหรับกำลังพลกองทัพบก ใช้สำหรับโทรศัพท์ภายในผ่านระบบ Internet โดยใช้หมายเลข 5 ตัวทบ. เชื่อมต่อกับระบบชุมสาย PBX ของกองทัพบก

## สิ่งที่ต้องเตรียม

1. **โทรศัพท์ Android** เวอร์ชัน 9.0 (Pie) ขึ้นไป
2. **การเชื่อมต่อ Internet** — Wi-Fi หรือ Mobile Data (3G/4G/5G)
3. **บัญชีผู้ใช้งาน** — Username และ Password ที่ได้รับจากหน่วย

> **หมายเหตุ:** หากยังไม่มีบัญชีผู้ใช้งาน กรุณาติดต่อ แผนกเครือข่ายภายใน กองการสื่อสาร ศูนย์โทรคมนาคมและเทคโนโลยีสารสนเทศ กรมการทหารสื่อสาร กองทัพบก

## ขั้นตอนเริ่มต้นอย่างรวดเร็ว

### 1. ติดตั้งแอป
ดาวน์โหลดไฟล์ APK จากช่องทางที่หน่วยกำหนด แล้วติดตั้งบนโทรศัพท์

### 2. เปิดแอปและอนุญาตสิทธิ์
เมื่อเปิดแอปครั้งแรก ระบบจะขออนุญาตสิทธิ์ต่าง ๆ:
- **ไมโครโฟน** — สำหรับการโทร (จำเป็น)
- **การแจ้งเตือน** — สำหรับรับสายเรียกเข้า (จำเป็น)
- **รายชื่อผู้ติดต่อ** — สำหรับแสดงชื่อผู้โทร (ไม่บังคับ)
- **กล้อง** — สำหรับวิดีโอคอล (ไม่บังคับ)

กรุณา **อนุญาตทั้งหมด** เพื่อให้แอปทำงานได้อย่างเต็มประสิทธิภาพ

### 3. เข้าสู่ระบบ
กรอก **Username** และ **Password** ที่ได้รับจากหน่วย แล้วกด **Login**

### 4. พร้อมใช้งาน
เมื่อเข้าสู่ระบบสำเร็จ จะแสดงหน้าหลักพร้อมเมนู:
- **Contacts** — รายชื่อผู้ติดต่อ
- **Calls** — ประวัติการโทร
- **Conversations** — ข้อความ

## ความปลอดภัย

RTAphone ใช้การเข้ารหัสการสื่อสารดังนี้:
- **TLS** — เข้ารหัสการเชื่อมต่อระหว่างแอปกับเซิร์ฟเวอร์
- **SRTP** — เข้ารหัสเสียงสนทนาระหว่างการโทร

การสนทนาทั้งหมดจะถูกเข้ารหัสจากต้นทางถึงปลายทาง
