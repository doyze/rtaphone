# นโยบายความเป็นส่วนตัว (Privacy Policy)

**RTAphone — ระบบโทรศัพท์ออนไลน์ กองทัพบก**

อัปเดตล่าสุด: มีนาคม 2569

---

## 1. ข้อมูลที่เก็บรวบรวม

RTAphone เก็บรวบรวมข้อมูลเฉพาะที่จำเป็นต่อการให้บริการโทรศัพท์เท่านั้น:

- **ข้อมูลบัญชี**: Username และ Password ที่ใช้เข้าสู่ระบบ (เก็บในเครื่องเท่านั้น)
- **ประวัติการโทร**: บันทึกหมายเลขที่โทรออก/รับสาย วันที่ และระยะเวลา (เก็บในเครื่องเท่านั้น)
- **รายชื่อผู้ติดต่อ**: รายชื่อที่ผู้ใช้เพิ่มเอง (เก็บในเครื่องเท่านั้น)
- **FCM Token**: สำหรับรับการแจ้งเตือนสายเรียกเข้า (ส่งไปยังเซิร์ฟเวอร์)

## 2. ข้อมูลที่ไม่เก็บรวบรวม

- ไม่บันทึกเนื้อหาการสนทนา
- ไม่เก็บข้อมูลตำแหน่ง (Location)
- ไม่เก็บข้อมูลส่วนบุคคลอื่น ๆ
- ไม่ส่งข้อมูลไปยังบุคคลที่สาม

## 3. การเข้ารหัสข้อมูล

- การเชื่อมต่อระหว่างแอปกับเซิร์ฟเวอร์ใช้ **TLS** (Transport Layer Security)
- เสียงสนทนาเข้ารหัสด้วย **SRTP** (Secure Real-time Transport Protocol)

## 4. การเก็บรักษาข้อมูล

- ข้อมูลทั้งหมดเก็บในเครื่องของผู้ใช้
- เมื่อถอนการติดตั้งแอป ข้อมูลทั้งหมดจะถูกลบ
- เซิร์ฟเวอร์เก็บเฉพาะข้อมูลการลงทะเบียน (Registration) เพื่อให้บริการโทรศัพท์

## 5. สิทธิ์ที่แอปขอ

| สิทธิ์ | เหตุผล |
|--------|--------|
| ไมโครโฟน | สำหรับการโทรด้วยเสียง |
| การแจ้งเตือน | สำหรับรับสายเรียกเข้า |
| รายชื่อผู้ติดต่อ | แสดงชื่อผู้โทร |
| กล้อง | วิดีโอคอล |
