## เกี่ยวกับ RTAphone

RTAphone เป็นแอปพลิเคชันโทรศัพท์ผ่านอินเทอร์เน็ต (VoIP/SIP) สำหรับกำลังพลกองทัพบก ใช้สำหรับโทรศัพท์ผ่านระบบเครือข่าย Internet โดยใช้หมายเลข 5 ตัวทบ. เชื่อมต่อกับระบบชุมสายโทรศัพท์ 5 ตัวทบ.

### คุณสมบัติหลัก

- โทรออก/รับสาย ผ่าน Internet ด้วยหมายเลข 5 ตัวทบ.
- การเข้ารหัสการสนทนา (SRTP/TLS) เพื่อความปลอดภัย
- รองรับ Push Notification (FCM) รับสายได้แม้ปิดแอป
- รองรับ Android 9.0 ขึ้นไป

### ข้อกำหนดเบื้องต้น

- โทรศัพท์ Android เวอร์ชัน 9.0 (Pie) ขึ้นไป
- การเชื่อมต่อ Internet (Wi-Fi หรือ Mobile Data)
- บัญชีผู้ใช้ (Username/Password) ที่ได้รับจากหน่วย
