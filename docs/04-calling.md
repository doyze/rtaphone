# การโทรออกและรับสาย

## การโทรออก

### วิธีที่ 1: กดหมายเลขโดยตรง
1. กดปุ่มโทรศัพท์ (มุมล่างขวา) เพื่อเปิดแป้นกดหมายเลข
2. กดหมายเลข 5 ตัวทบ. (เช่น `2001`)
3. กดปุ่ม **โทรออก** (สีเขียว)

### วิธีที่ 2: โทรจากรายชื่อผู้ติดต่อ
1. ไปที่ **Contacts** > เลือกผู้ติดต่อ > กด **โทรออก**

### วิธีที่ 3: โทรจากประวัติการโทร
1. ไปที่ **Calls** > กดที่รายการที่ต้องการโทรซ้ำ

## การรับสาย
- ปุ่ม **สีเขียว** — รับสาย
- ปุ่ม **สีแดง** — ปฏิเสธสาย

## ระหว่างการโทร

| ปุ่ม | การทำงาน |
|------|----------|
| ปิดเสียง (Mute) | ปิด/เปิดไมโครโฟน |
| ลำโพง (Speaker) | สลับเสียงไปลำโพงนอก |
| พัก (Pause) | พักสายชั่วคราว |
| โอนสาย (Transfer) | โอนสายไปหมายเลขอื่น |
| แป้นกด (Numpad) | เปิดแป้นกด DTMF |
| วางสาย | วางสาย (สีแดง) |

## การโอนสาย
1. ระหว่างสนทนา กด **Transfer**
2. กดหมายเลขปลายทาง
3. กด **โอนสาย**
