# การมีส่วนร่วมและการแปลภาษา

## การแปลภาษา

RTAphone รองรับหลายภาษา หากต้องการช่วยแปลหรือปรับปรุงข้อความในแอป สามารถส่งคำแนะนำได้ที่:

- **GitHub Issues**: [github.com/doyze/rtaphone/issues](https://github.com/doyze/rtaphone/issues)

## การรายงานปัญหา (Bug Report)

หากพบปัญหาการใช้งาน สามารถรายงานได้ 2 ช่องทาง:

### 1. ผ่านแอป
1. ไปที่ **☰** > **Help** > **Troubleshooting**
2. กด **Send logs**
3. แจ้งรายละเอียดปัญหาให้แผนกเครือข่ายภายใน

### 2. ผ่าน GitHub
เปิด Issue ที่ [github.com/doyze/rtaphone/issues](https://github.com/doyze/rtaphone/issues) พร้อมระบุ:
- รุ่นโทรศัพท์
- เวอร์ชัน Android
- เวอร์ชัน RTAphone
- รายละเอียดปัญหา

## การพัฒนาต่อยอด

RTAphone เป็น Open Source ภายใต้ GPLv3 ผู้ที่สนใจพัฒนาต่อยอดสามารถ Fork และส่ง Pull Request ได้ที่:
- [github.com/doyze/rtaphone](https://github.com/doyze/rtaphone)
