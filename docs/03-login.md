# การเข้าสู่ระบบ

## เข้าสู่ระบบครั้งแรก
เมื่อเปิดแอป RTAphone ครั้งแรก จะแสดงหน้าจอ **RTA SIP Account**

### ขั้นตอน
1. กรอก **Username** — หมายเลข 5 ตัวทบ. (เช่น `2003`)
2. กรอก **Password** — รหัสผ่านที่ได้รับจากหน่วย
3. กด **Login**

> **หมายเหตุ:** ไม่ต้องกรอก Domain หรือ Server — ระบบจะเชื่อมต่อกับเซิร์ฟเวอร์ของกองทัพบกโดยอัตโนมัติ

### เมื่อเข้าสู่ระบบสำเร็จ
- แอปจะแสดงหน้าหลัก พร้อมรับสายและโทรออกได้ทันที

### เมื่อเข้าสู่ระบบไม่สำเร็จ
- **Username หรือ Password ผิด** — ตรวจสอบตัวพิมพ์เล็ก/ใหญ่
- **ไม่มี Internet** — ตรวจสอบ Wi-Fi หรือ Mobile Data
- **บัญชีถูกระงับ** — ติดต่อแผนกเครือข่ายภายใน

## การออกจากระบบ
1. กดเมนู **☰** > **Settings** > เลือกบัญชี > **Delete account**

## การเข้าสู่ระบบอัตโนมัติ
หลังจากเข้าสู่ระบบครั้งแรก แอปจะจดจำบัญชีไว้ ไม่ต้องกรอกใหม่
